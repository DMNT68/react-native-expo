# ⚛️ React Native con Expo

[![React Native](https://img.shields.io/badge/React_Native-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://reactnative.dev/)
[![Expo](https://img.shields.io/badge/Expo-000020?style=for-the-badge&logo=expo&logoColor=white)](https://expo.dev/)

React Native es un framework que permite desarrollar aplicaciones móviles nativas para iOS y Android usando JavaScript y React. Expo es una plataforma que simplifica el desarrollo de aplicaciones con React Native, proporcionando herramientas integradas y servicios que aceleran el desarrollo y desplegamiento.

## Descripción del Proyecto

Este repositorio contiene el código y los recursos desarrollados durante el curso de React Native con Expo de Fernando Herrera. El objetivo principal es aprender a construir aplicaciones móviles modernas y eficientes utilizando React Native y Expo.
